Build Python3.7 module of OpenVDB (Windows10 x64) 1. Install Python3.7 1.1 Download "Windows x86-64 executable installer" and Install to "C:\Python37" https://www.python.org/downloads/release/python-373/ 1.2 Install numpy by pip cd C:\Python37\Scripts pip3 install numpy 2. Install vcpkg tag: 2020.04 to C:\dev cd c:\dev git clone https://github.com/microsoft/vcpkg.git -b 2020.04 cd vcpkg bootstrap-vcpkg.bat vcpkg install zlib:x64-windows vcpkg install blosc:x64-windows vcpkg install openexr:x64-windows vcpkg install tbb:x64-windows vcpkg install boost-iostreams:x64-windows vcpkg install boost-system:x64-windows vcpkg install boost-any:x64-windows vcpkg install boost-algorithm:x64-windows vcpkg install boost-uuid:x64-windows vcpkg install boost-interprocess:x64-windows vcpkg install pybind11:x64-windows cd c:\dev\vcpkg\downloads\tools\python\python-3.7.3-x64 curl https://bootstrap.pypa.io/get-pip.py -O get-pip.py python get-pip.py cd Scripts pip3 install numpy cd c:\dev\vcpkg set PATH=c:\dev\vcpkg\downloads\tools\python\python-3.7.3-x64;%PATH% vcpkg install boost-python:x64-windows 3. Download and Install Cmake https://cmake.org/download/ 4. Building OpenVDB 4.1 Download OpenVDB from GitHub git clone https://github.com/AcademySoftwareFoundation/openvdb.git 4.2 Change NUMPY option to ON cd openvdb\openvdb\openvdb\python notepad CMakeList.txt option(USE_NUMPY [=[ Build the python library with numpy support. Currently requires CMake 3.14.]=] ON) option(OPENVDB_PYTHON_WRAP_ALL_GRID_TYPES [=[ Expose (almost) all of the grid types in the python module. Otherwise, only FloatGrid, BoolGrid and Vec3SGrid will be exposed (see, e.g., exportIntGrid() in python/pyIntGrid.cc). Compiling the Python module with this ON can be very memory-intensive.]=] ON) • Change the USE_NUMPY and OPENVDB_PYTHON_WRAP_ALL_GRID_TYPES option in CMakeList.txt to ON. 4.3 Build Python module of OpenVDB mkdir build cd build cmake -DCMAKE_TOOLCHAIN_FILE=c:\dev\vcpkg\scripts\buildsystems\vcpkg.cmake -DVCPKG_TARGET_TRIPLET=x64-windows -A x64 -DOPENVDB_BUILD_PYTHON_MODULE=ON .. ... -- ---------------------------------------------------- -- ------------ Configuring OpenVDBPython ------------- -- ---------------------------------------------------- -- Found Python: C:/dev/vcpkg/installed/x64-windows/include/python3.7) (found suitable version "3.7.3", minimum required is "2.7") -- Found NumPy: C:/Python37/Lib/site-packages/numpy/core/include (found suitable version "1.19.2", minimum required is "1.12.1") -- Found boost_python37 -- Found boost_numpy37 • You will find boost_python37 and boost_numpy37 cmake --build . --parallel 4 --config Release cd openvdb\openvdb\python\Release dir boost_numpy37-vc142-mt-x64-1_72.dll boost_python37-vc142-mt-x64-1_72.dll Half-2_3.dll pyopenvdb.exp pyopenvdb.lib pyopenvdb.pyd python37.dll tbb.dll • You will need other dependency DLL(bloc.dll, lz4.dll, openvdb.dll, zlib.dll, zstd.dll), these DLL find in the vcpkg and openvdb folder.
